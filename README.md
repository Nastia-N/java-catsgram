# Catsgram
Социальная сеть для публикации фото и постов о котах с возможностью загрузки изображений и управления контентом.

## Возможности
##### Пользователи
Регистрация пользователей — создание учетной записи с email, паролем  
Управление профилем — обновление username, email, пароля  
Поиск пользователей — просмотр всех зарегистрированных пользователей  
Уникальные email — проверка на уникальность email при регистрации

##### Посты
Создание постов — публикация постов с описанием  
Обновление постов — редактирование описания публикаций  
Просмотр постов — получение постов по ID с информацией об авторе  
Автоматические даты — время создания поста фиксируется автоматически

##### Изображения
Загрузка изображений — прикрепление изображений к постам  
Множественная загрузка — возможность загружать несколько изображений за раз  
Скачивание изображений — загрузка изображений в бинарном формате  
Хранение на файловой системе — изображения сохраняются в указанной директории  
Уникальные имена файлов — генерация уникальных имен для предотвращения конфликтов

## Эндпоинты
##### Пользователи (/users)
POST   /users                    - создать пользователя  
PUT    /users/{userId}           - обновить пользователя  
GET    /users                    - получить всех пользователей  
GET    /users/{userId}           - получить пользователя по ID  

##### Посты (/posts)
POST   /posts                    - создать пост  
GET    /posts/{postId}           - получить пост по ID  
PUT    /posts/{postId}           - обновить пост  

##### Изображения (/posts/{postId}/images)
POST   /posts/{postId}/images    - загрузить изображения к посту  
GET    /posts/{postId}/images    - получить все изображения поста  
GET    /posts/{postId}/images/{imageId} - скачать изображение (бинарный поток)

## Архитектура
Слоистая архитектура (Controller-Service-Repository)  
REST API с JSON  
DTO паттерн с мапперами  
Кастомные исключения с глобальной обработкой

## Особенности реализации
Базовая реализация репозитория (BaseRepository) с общими CRUD-операциями  
Работа с файловой системой для хранения изображений  
Валидация бизнес-правил на уровне сервисов  
Поддержка загрузки мультимедийных файлов

## Технологический стек
Java 21 + Spring Boot 3  
Spring MVC (REST API)  
Spring JDBC для работы с базой данных  
PostgreSQL / H2 Database  
Lombok
